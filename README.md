# 📝 Intern Task Tracker

## 📌 Description
Intern Task Tracker is a Java-based project used to manage and track tasks assigned to interns. It helps in organizing tasks, updating their status, and keeping records efficiently.

---

## 🚀 Features
- ➕ Add new tasks
- 📋 View all tasks
- ✏️ Update task details
- ❌ Delete tasks
- 📊 Track task status (Pending / Completed)

---

## 🛠️ Tech Stack
- Java
- OOP Concepts
- JDBC
- (Optional: File Handling / MySQL if you used it)

---

## ⚙️ Functionality
- Create Task
- Read Task List
- Update Task
- Delete Task

---

1. Clone the repository  
2. Open project in VS Code / any IDE
